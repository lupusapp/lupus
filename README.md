# Lupus

## Build
```sh
$ git clone https://github.com/lupusapp/lupus
$ cd lupus
$ git clone https://github.com/TokTok/c-toxcore
$ cd c-toxcore
$ mkdir _build && cd _build
$ cmake ..
$ make
$ cd ../..
$ mkdir build && cd build
$ cmake ..
$ make
$ ./Lupus
```
